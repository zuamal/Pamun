# F25 — Semantic Zoom (줌 레벨별 노드/Edge 보정)

## 개요

줌 아웃 시 텍스트가 뭉개지고 Edge가 사라지듯 얇아지는 문제를 해소한다.
두 가지 보정을 적용한다:

1. **노드 역보정(Counter-scale):** zoom < 0.9 시 노드 내부에 `scale(1/zoom)`을 적용하여 텍스트 가시성 유지
2. **Edge 두께 보정:** zoom < 0.9 시 `strokeWidth = max(2, 기본값 / zoom)`으로 최소 두께 보장

---

## 완료 기준

### `graphStore.ts` — zoomLevel 상태 추가

- [ ] `zoomLevel: number` 추가 (초기값: `1`)
- [ ] `setZoomLevel: (z: number) => void` 추가
- [ ] `onViewportChange` 콜백에서 throttle(16ms) 적용 권장 (매 프레임 store 업데이트 방지)

### `RequirementGraph.tsx` — viewport 감지

- [ ] `onViewportChange={(vp) => setZoomLevel(vp.zoom)}` 연결

### `RequirementNode.tsx` — 역보정 + 콘텐츠 분기

- [ ] `graphStore`에서 `zoomLevel` 구독
- [ ] zoom < 0.9 시 노드 내부 래퍼에 역보정 적용:
  ```tsx
  const scale = zoomLevel < 0.9 ? Math.min(1 / zoomLevel, 1.8) : 1
  <div style={{ transform: `scale(${scale})`, transformOrigin: 'top left' }}>
  ```
- [ ] 줌 레벨별 콘텐츠 분기:

  | 조건 | 표시 내용 |
  |------|-----------|
  | zoom ≤ 0.3 | 색상 도트만 (라벨·제목·연결 수 모두 숨김) |
  | 0.3 < zoom < 0.9 | 색상 도트 + ID 라벨만 (제목·연결 수 숨김) |
  | 0.9 ≤ zoom ≤ 1.2 | 라벨 + 제목 1줄 (`line-clamp-1`) |
  | zoom > 1.2 | 라벨 + 제목 2줄 (`line-clamp-2`) + 연결 수 뱃지 |

- [ ] zoom ≤ 0.3 구간에서 역보정 상한(1.8)이 적용되므로 도트가 일정 크기 이상으로 보정된다. 별도 처리 불필요
- [ ] 역보정 적용 시 노드 컨테이너 크기는 고정(220×80px) 유지. 내부 콘텐츠만 스케일 변환

### Edge 커스텀 컴포넌트 — 두께 보정

- [ ] `zoomLevel`을 `graphStore`에서 구독
- [ ] zoom < 0.9 시 strokeWidth 및 화살표 마커 크기 보정:
  ```ts
  const baseWidth = isImpact ? 3 : 2  // approved/impact 기본값
  const strokeWidth = zoomLevel < 0.9
    ? Math.max(2, baseWidth / zoomLevel)
    : baseWidth

  // 화살표 마커 크기 (markerWidth / markerHeight)
  const baseMarker = 8  // 기본 화살표 크기
  const markerSize = zoomLevel < 0.9
    ? Math.max(8, baseMarker / zoomLevel)
    : baseMarker
  ```
- [ ] React Flow `markerEnd` / `markerStart`에 `markerWidth`, `markerHeight`를 `markerSize`로 주입
- [ ] zoom ≥ 0.9 시 현행 두께 및 화살표 크기 유지 (approved: 2px, 마커: 8)
- [ ] strokeDasharray(점선) 패턴은 줌 레벨 무관하게 유지

### 성능

- [ ] `setZoomLevel` 호출에 throttle(16ms) 또는 `requestAnimationFrame` 적용
- [ ] `RequirementNode`에서 `zoomLevel`을 `useShallow` 또는 selector로 구독하여 불필요한 리렌더링 방지

### 타입 체크

- [ ] `npx tsc --noEmit` 오류 없음

## 주의사항

- `transformOrigin: 'top left'`로 설정해야 노드가 좌측 상단 기준으로 스케일되어 위치 오차가 없다
- 역보정 상한 `1.8`은 zoom ≈ 0.55 이하에서 노드가 지나치게 커지는 것을 방지하는 값이다. 실제 테스트 후 조정 가능
- 임계값(0.3 / 0.9 / 1.2)은 구현 후 미세 조정 가능
