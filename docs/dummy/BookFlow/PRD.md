# BookFlow — 온라인 서점 플랫폼 PRD

## 1. 개요

BookFlow는 도서 검색, 주문, 리뷰를 통합 제공하는 온라인 서점 플랫폼이다. 사용자는 도서를 탐색하고 장바구니에 담아 결제할 수 있으며, 구매한 도서에 대해 리뷰를 작성할 수 있다.

## 2. 사용자 인증

### 2.1 회원가입

사용자는 이메일과 비밀번호로 회원가입할 수 있다. 비밀번호는 최소 8자 이상이어야 하며, 영문 대소문자와 숫자를 각각 1개 이상 포함해야 한다. 회원가입 시 이메일 인증을 완료해야 계정이 활성화된다.

### 2.2 로그인 및 세션 관리

사용자는 이메일과 비밀번호로 로그인한다. 로그인 성공 시 JWT 토큰을 발급하며, 액세스 토큰의 유효기간은 1시간, 리프레시 토큰의 유효기간은 7일이다. 5회 연속 로그인 실패 시 계정이 30분간 잠긴다.

## 3. 도서 관리

### 3.1 도서 검색

사용자는 제목, 저자, ISBN으로 도서를 검색할 수 있다. 검색 결과는 관련도순, 출판일순, 가격순으로 정렬할 수 있다. 검색어 자동완성 기능을 제공하여 사용자가 빠르게 원하는 도서를 찾을 수 있도록 한다.

### 3.2 도서 상세 페이지

도서 상세 페이지에는 제목, 저자, 출판사, 출판일, 가격, 재고 상태, 도서 설명, 독자 리뷰 목록이 표시된다. 재고가 0인 도서는 "품절" 상태로 표시되며, 사용자가 입고 알림을 신청할 수 있다.

### 3.3 카테고리 및 필터

도서는 대분류(문학, 경영, IT, 자기계발 등)와 소분류로 분류된다. 사용자는 카테고리, 가격 범위, 출판일 범위, 평점 필터를 조합하여 도서를 탐색할 수 있다.

## 4. 주문 및 결제

### 4.1 장바구니

사용자는 도서를 장바구니에 추가, 수량 변경, 삭제할 수 있다. 장바구니에 담긴 도서의 재고가 부족해지면 사용자에게 알림을 표시한다. 비회원도 장바구니를 이용할 수 있으며, 로그인 시 비회원 장바구니가 회원 장바구니에 병합된다.

### 4.2 결제 프로세스

결제는 신용카드, 간편결제(카카오페이, 네이버페이), 무통장입금을 지원한다. 주문 시 배송지 정보를 입력하며, 최근 배송지 3개를 저장하여 재사용할 수 있다. 결제 완료 후 주문 확인 이메일을 발송한다.

### 4.3 주문 상태 관리

주문 상태는 결제완료 → 배송준비 → 배송중 → 배송완료 순서로 변경된다. 결제완료 상태에서만 주문 취소가 가능하며, 배송완료 후 7일 이내에 반품을 신청할 수 있다. 각 상태 변경 시 사용자에게 이메일 또는 푸시 알림을 발송한다.

## 5. 리뷰 시스템

### 5.1 리뷰 작성

구매 확정된 도서에 대해서만 리뷰를 작성할 수 있다. 리뷰는 1~5점 평점과 텍스트로 구성되며, 텍스트는 최소 20자 이상이어야 한다. 한 사용자가 동일 도서에 대해 리뷰를 하나만 작성할 수 있으며, 작성 후 수정은 가능하지만 삭제 후 재작성은 불가하다.

### 5.2 리뷰 표시 및 정렬

도서 상세 페이지에서 리뷰는 최신순, 평점 높은순, 평점 낮은순으로 정렬할 수 있다. 전체 평균 평점과 평점 분포(1~5점 각 비율)를 도서 상세 페이지 상단에 표시한다. 비속어가 포함된 리뷰는 자동으로 필터링되어 관리자 검토 대기 상태로 전환된다.
