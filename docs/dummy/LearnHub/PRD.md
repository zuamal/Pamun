# LearnHub — 사내 교육 LMS PRD

## 1. 개요

LearnHub는 사내 직원 교육을 위한 학습 관리 시스템(LMS)이다. 관리자가 교육 과정을 생성하고, 직원이 온라인으로 수강하며, 학습 진도와 평가 결과를 추적할 수 있다.

## 2. 사용자 역할

### 2.1 역할 정의

시스템에는 세 가지 역할이 있다: 관리자(Admin), 강사(Instructor), 수강생(Learner). 관리자는 과정 생성, 사용자 관리, 통계 조회 권한을 가진다. 강사는 자신이 담당하는 과정의 콘텐츠 관리와 평가 출제 권한을 가진다. 수강생은 배정된 과정을 수강하고 평가에 응시할 수 있다.

### 2.2 SSO 연동

사내 시스템이므로 별도 회원가입 없이 기존 사내 SSO(SAML 2.0)를 통해 인증한다. SSO 로그인 시 사번, 이름, 부서, 직급 정보가 자동으로 동기화된다. 역할은 관리자가 수동으로 부여하며, 기본값은 수강생이다.

## 3. 과정 관리

### 3.1 과정 생성

관리자 또는 강사가 교육 과정을 생성한다. 과정에는 제목, 설명, 카테고리, 수강 기간, 담당 강사를 설정한다. 과정은 공개(모든 직원 수강 가능)와 지정(특정 부서나 직원에게만 배정) 두 가지 유형이 있다.

### 3.2 콘텐츠 구성

과정은 여러 개의 모듈로 구성되고, 각 모듈은 여러 개의 레슨으로 구성된다. 레슨 유형: 동영상, 문서(PDF 업로드), 텍스트(리치 텍스트 에디터). 모듈 간 순서를 지정할 수 있으며, 이전 모듈을 완료해야 다음 모듈에 접근할 수 있는 순차적 학습 모드를 선택적으로 활성화할 수 있다.

### 3.3 수강생 배정

지정 과정의 경우 관리자가 개별 직원 또는 부서 단위로 수강생을 배정한다. 배정 시 수강 기한을 설정할 수 있다. 수강생에게 배정 알림 이메일이 발송된다. 배정 후 수강생이 기한 내에 미수강 시 리마인더 이메일이 발송된다.

## 4. 학습 및 진도

### 4.1 수강

수강생은 배정된 과정의 레슨을 순서대로 학습한다. 동영상 레슨은 전체 재생 시간의 90% 이상을 시청해야 완료로 인정한다. 문서 레슨은 열람 후 완료 버튼을 클릭하면 완료로 인정한다. 학습 중단 시 마지막 시청/열람 위치를 기록하여 이어보기를 지원한다.

### 4.2 진도 추적

수강생별 과정 진도율(완료 레슨 수 / 전체 레슨 수)을 실시간으로 계산한다. 관리자와 강사는 과정별 전체 수강생의 진도 현황을 대시보드에서 조회할 수 있다. 진도율이 수강 기한 대비 뒤처지는 수강생에게 자동 리마인더를 발송한다.

## 5. 평가

### 5.1 퀴즈 출제

강사는 과정 또는 모듈 단위로 퀴즈를 출제할 수 있다. 퀴즈 유형: 객관식(단일선택, 다중선택), 주관식(단답형). 각 문제에 배점을 설정하며, 합격 기준 점수를 과정 단위로 지정한다.

### 5.2 응시 및 채점

수강생은 해당 모듈 또는 과정의 학습을 완료한 후 퀴즈에 응시할 수 있다. 객관식은 자동 채점되며, 주관식은 강사가 수동 채점한다. 응시 횟수 제한을 설정할 수 있으며, 재응시 시 이전 점수는 보존된다. 합격 기준 미달 시 과정 이수가 불인정된다.

### 5.3 수료증

과정의 모든 모듈을 완료하고 퀴즈에서 합격 기준을 충족하면 수료증이 자동 발급된다. 수료증에는 수강생 이름, 과정명, 수료일, 고유 인증번호가 포함된다. 수료증은 PDF로 다운로드할 수 있다.

## 6. 통계 및 리포트

### 6.1 관리자 대시보드

관리자는 전체 과정의 수강 현황, 완료율, 평균 퀴즈 점수를 조회할 수 있다. 부서별, 과정별, 기간별 필터를 제공한다. 미수강자 목록과 수강 기한 임박자 목록을 확인할 수 있다.

### 6.2 수강생 학습 이력

수강생은 자신의 학습 이력(수강한 과정, 진도율, 퀴즈 점수, 수료증)을 조회할 수 있다. 관리자는 특정 수강생의 전체 학습 이력을 조회할 수 있다.
